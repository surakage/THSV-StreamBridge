# THSV StreamBridge 2.1.1

THSV StreamBridge 2.1.1 is a security, reliability, add-on, and fresh-install maintenance release.

## Highlights

- Adds Random Clip Player 1.4.0 as a separately downloadable optional add-on with no-repeat rotation, playback-aware scheduling, a four-second fade buffer, and creator-triggerable Enable/Disable actions.
- Publishes every valid package under `addons/` as an independent, checksummed, provenance-attested `.thsv-addon` Release asset instead of silently bundling optional features into core.
- Hardens Streamer.bot and add-on relay authentication, package review/install integrity, module capability revocation, browser-origin checks, regex blocker validation, installer paths, durable delivery, deduplication persistence, and generated package path confinement.
- Makes Windows startup single-instance safe and opens the setup wizard through the configured custom port only after confirming the expected healthy StreamBridge service.
- Updates the published normalized-event JSON Schema to match runtime presentation fields, bounds, JSON values, and stable-event identity requirements.

## Acceptance evidence

- 66 Vitest files and 468 unit/integration tests pass.
- All 6 Playwright wizard and overlay checks pass.
- Typecheck, lint, production build, configuration validation, npm audit, and Windows custom-port lifecycle tests pass.
- The self-contained Windows archive and Random Clip Player add-on both build with adjacent SHA-256 files; GitHub Actions publishes provenance attestations and the production SBOM.

## Install

Download `THSV-StreamBridge-2.1.1.zip` and `THSV-StreamBridge-2.1.1.zip.sha256`, verify them using `RELEASE-VERIFICATION.md`, extract the ZIP, and run `Install THSV StreamBridge.cmd`.

Random Clip Player is optional. Download its `.thsv-addon` and adjacent checksum from this Release, then install it from the authenticated Wizard's Add-ons page. Import its included Streamer.bot package only if you choose to use the add-on.

Normal upgrades and default uninstall preserve creator configuration, secrets, uploads, logs, backups, installed add-ons, and add-on state.
