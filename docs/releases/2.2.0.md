# THSV StreamBridge 2.2.0

Version 2.2.0 completes the first public add-on suite while keeping core and every optional feature independently installable.

## Included in core

- Local-first Streamer.bot event bridge for Twitch, YouTube, Kick, and TikTok through TikFinity.
- Unified chat, commands, alerts, timed actions, rewards, and browser overlays.
- Durable delivery, deduplication, retry/dead-letter handling, authenticated management, and safe add-on capability boundaries.
- Guided setup wizard with clean add-on settings, action grants, preview links, and a manual update checker.

## Separate add-on downloads

- Random Clip Player 1.4.0
- Automated Shoutouts 1.0.0
- User Translate 1.0.0
- Auto Translate 1.0.0
- Ko-fi Donations 1.0.1
- Subathon Timer 1.1.0

Each add-on ZIP contains the wizard-installable package, its own Streamer.bot `.sb` import, checksum, and setup notes. Optional add-on actions are not mixed into the core import set.

## Before going live

Install core first, connect and inspect Streamer.bot, then install only the add-ons you want. Run simulated or manual acceptance checks before enabling live sending, translation, financial alerts, shoutouts, or timer mutation.
