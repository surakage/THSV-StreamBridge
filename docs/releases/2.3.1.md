# THSV StreamBridge 2.3.1

Version 2.3.1 is the recommended download for the 2.3 creator-automation release.

It includes every feature and separate add-on from 2.3.0, plus a Windows installer reliability fix discovered by the parallel GitHub Actions acceptance run.

## Fixed

- Portable upgrades now retry transient Windows file locks while replacing the bundled runtime, launcher, application directories, rollback directories, and atomic installation records.
- Retry behavior is bounded and applies only to Windows-style transient filesystem errors. Permanent failures still stop installation and preserve rollback behavior.
- The full installer upgrade, downgrade, data-preservation, tamper-rejection, and uninstall acceptance path passed five consecutive local Windows runs before publication.
