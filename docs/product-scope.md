# Revised product scope

THSV StreamBridge 2 is a modular Streamer.bot automation framework for creators broadcasting to Twitch, YouTube, Kick, and TikTok through TikFinity.

The revised core contains platform configuration, Multi-Chat, Multi-Commands, Multi-Alerts, Multi-Timed Actions, chat filters, Twitch Channel Point Reward integration, Kick Channel Reward integration, the setup/management wizard, diagnostics, installation, backup, export, updates, and the module host.

The core does not contain viewer points, progression, XP, levels, inventories, economy, companions, goals, stream labels, games, competitive systems, or AI engagement. Speaker.bot orchestration is also outside the exclusive core scope. Existing implementations are preserved as archived future add-on candidates until they can depend solely on public module contracts.

Facebook is unsupported. Twitch, YouTube, and Kick use documented Streamer.bot integration surfaces; TikTok uses TikFinity with unverified fields and limitations reported honestly.

Core must start and remain healthy with no optional add-ons installed. Add-on failure must degrade only that add-on, never platform intake or unrelated built-in modules.
