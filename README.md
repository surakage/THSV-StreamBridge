# THSV StreamBridge

THSV StreamBridge is a modular, local-first livestream automation framework for normalizing and routing events across streaming platforms and creator tools.

This repository is a clean rebuild. It does not depend on earlier Streamer.bot, Speaker.bot, overlay, chatbot, or JSON projects.

## Status

Repository scaffold only. Milestone 1 (Bridge Core) has not yet been implemented.

## Planned stack

- Node.js LTS
- Strict TypeScript
- Runtime schema validation
- WebSocket and HTTP diagnostics support
- PowerShell scripts for Windows

## Security

Never commit credentials or local runtime data. Copy `.env.example` to `.env` only when local secrets are needed.
